# JSDoc Auto Import

Automates JSDoc `@import` type management using modern TypeScript 5.5+ syntax.

## Features

- **Auto-import Types:** Suggests types from your workspace directly inside JSDoc comments.
- **Top-of-File Blocks:** Groups imports at the top of the file automatically.
- **Convert Inline Imports:** Convert `import('...').Type` into `@import` blocks easily via Command Palette.
- **Organize Imports:** Sort and clean up unused JSDoc imports.

## Requirements

- VS Code 1.107.0 or higher
- TypeScript 5.5+ syntax support
